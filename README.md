# Tarea 3 — Cálculo de Sueldo de Empleado

## Descripción
Este proyecto consiste en una aplicación de escritorio que permite calcular el sueldo de un empleado a partir del pago por hora y la cantidad de horas trabajadas.

El sistema realiza automáticamente los cálculos necesarios para determinar el sueldo bruto, los descuentos correspondientes y el sueldo neto final.

Los resultados se muestran en la interfaz del programa utilizando controles de tipo **Label**, permitiendo visualizar cada cálculo de forma clara.

---

## Funcionalidades
El programa realiza los siguientes cálculos:

- **Sueldo bruto**
- **Descuento AFP (2.87%)**
- **Descuento SFS (3.04%)**
- **Descuento adicional de 700 pesos**
- **Total de descuentos**
- **Sueldo neto**

---

## Tecnologías Utilizadas
- **Lenguaje:** C#
- **Framework:** .NET (Windows Forms)
- **Entorno de desarrollo:** Visual Studio 2022

---

## Uso o Ejecución
Para ejecutar el proyecto se deben seguir los siguientes pasos:

1. Clonar el repositorio desde GitHub.
2. Abrir el archivo `.sln` en **Visual Studio 2022**.
3. Compilar el proyecto.
4. Ejecutar la aplicación presionando **F5** o el botón **Iniciar**.
5. Ingresar el pago por hora y las horas trabajadas para obtener el cálculo del sueldo.
